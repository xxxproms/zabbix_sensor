# Общий перечень комплектующих (BOM) узла «Zabbix + Arduino + DS18B20»

Сводная таблица для **одного** готового устройства (одна плата Arduino + Ethernet + линия датчиков). Количество DS18B20 — **1–4** по проекту.

| № | Компонент | Кол-во | Назначение |
|---|-----------|--------|------------|
| 1 | Плата **Arduino Uno** или **Nano** (ATmega328P) | 1 | Контроллер, прошивка агента |
| 2 | Модуль **ENC28J60** (Ethernet SPI) | 1 | Доступ к сети, библиотека **UIPEthernet** |
| 3 | **DS18B20** | 1–4 | Температура, шина 1-Wire |
| 4 | Резистор **4.7 kΩ** | 1 | Подтяжка линии **DQ** (на всю шину 1-Wire) |
| 5 | Кабель **Ethernet** (Cat5e/Cat6), розетка/патч | 1 | Связь с коммутатором в **10.10.0.0/23** |
| 6 | Питание **5 В** для Arduino | 1 | USB с ПК/БП или стабилизированный адаптер **5 В** |
| 7 | Провод монтажный, припой, по желанию термоусадка | по месту | Соединения DS18B20 ↔ Arduino |
| 8 | По желанию: макетная плата, клеммники, корпус | — | Механика и удобство |

### Не электроника, но нужно для ввода в эксплуатацию

| Что | Зачем |
|-----|--------|
| ПК с **Arduino IDE 2** | Сборка и загрузка скетча |
| Учётная запись **Zabbix** (server/proxy **7.4.x**) | Хост, элементы, опрос **TCP 10050** |
| Уникальный **IP** в диапазоне **10.10.0.30–10.10.1.254** / маска **255.255.254.0** | Прописать в скетче (`USE_STATIC_IP`) |
| Уникальный **MAC** на каждую плату | Массив `mac[]` в скетче |
